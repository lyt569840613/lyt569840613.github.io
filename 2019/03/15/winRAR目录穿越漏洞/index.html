winRAR目录穿越漏洞

发表于 2019-03-15

1.漏洞编号 CVE-2018-20250

2.影响版本

WinRAR < 5.70 Beta 1

Bandizip < = 6.2.0.0

好压(2345压缩) < = 5.9.8.10907

360压缩 < = 4.0.0.1170

3.漏洞复现环境

角色 系统版本 IP 备注
靶机 Windows7（64bit） 192.168.43.231 安装WinRAR版本为5.6.1，关闭防火墙
攻击机 Kali Linux 64 Bit 2019.1a 192.168.43.242 Metasploit版本为5.0.2
攻击机 WIndows10（64bit） 安装Python3.7.2

exp下载：https://github.com/WyAtu/CVE-2018-20250/

4.漏洞复现过程

4.1 使用Metasploit5生成免杀Payload

4.2使用exp生成恶意程序

将exp.py中的evil_filename = “calc.exe”改为”WinRAR.exe”

将CVE-2018-20250文件夹复制到Windows10中，执行

python exp.py

生成恶意程序test.rar

4.3 在靶机上运行恶意程序

将test.rar 投放到win7上 解压运行

4.4 在kali上监听metasploit
4.5控制靶机

win7重启之后 kali会接受到会话 此时执行shell

进入到win7系统的system权限
