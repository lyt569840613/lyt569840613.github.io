如何利用GITHUB+Hexo搭建个人博客

发表于 2019-04-14

1.安装环境

1.安装Node.js：在Node.js官网下载最新版本

2.安装Git for Windows：在git官网下载，选择Windows

3.安装cmder：在cmder官网下载，Git Bash并不好用，所以选择使用cmder

4.配置环境变量：将下载好的cmder git Node.js 找到安装路径 右键 我的电脑–属性 –点击左边高级系统设置–点击环境变量 下方系统变量种找到path 点击编辑 然后新建添加环境变量 把三个环境变量都添加之后点击确定

2.创建Github

1.去github官网创建一个账号（GitHub是一个面向开源及私有软件项目的托管平台）

2.新建一个用户名为 .github.io的仓库 例如 我的github用户名叫lyt569840613 那么我创建的仓库名就是lyt569840613.github.io，将来的博客地址就是https://lyt569840613.github.io
